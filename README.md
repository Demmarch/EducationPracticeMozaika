# Ветка sqtcp/csocket. Что это?

Ветка, в которой бэкэнд написан на C++ с использованием QTcpSocket, а фронтенд на Dart с Flutter. Это и есть изначальная версия, написанная во время учебной практики.

## Из чего состоит?

Состоит из сервера, который написан на C++ с фреймворком Qt, и фронтендом (интерфейсом) на Dart с Flutter.

## Почему именное такие фреймворки?

Банально мой личный интерес к этим двум фреймворкам.
